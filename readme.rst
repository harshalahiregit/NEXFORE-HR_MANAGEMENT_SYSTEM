This project is a simple HR management system built using **CodeIgniter 3**.  
It allows managing employees and departments with full CRUD operations and provides **REST API endpoints** for external access.

The system includes:

- Admin dashboard showing total departments, total employees, and employees per department
- Employee management (Add, Edit, Delete, List)
- Department management (Add, Edit, Delete, List)
- Simple search/filter for employees
- REST API endpoints in JSON format

*******************
Server Requirements
*******************

- PHP version 7.0 or newer (Tested on PHP 7.4)
- MySQL
- Apache (XAMPP)
- CodeIgniter 3
